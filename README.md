ReCiclo - Protótipo de Baixa Fidelidade
Integrantes
Luiz Guilherme, Cauã Cordeiro, João Vitor Pedrosa, Silas Caleb
Proposta de Valor
O aplicativo ReCiclo tem como objetivo conectar pessoas que desejam doar itens com outras que precisam, promovendo sustentabilidade, reutilização e redução de desperdício.

Descrição das Telas
Home / Feed Exibe os itens disponíveis próximos ao usuário, com imagem, nome e distância.

Filtros Permite refinar a busca por categoria, distância e estado de conservação.

Detalhes do Item Mostra informações completas do item e permite entrar em contato com o doador.

Cadastro de Item Tela onde o usuário pode cadastrar um novo item para doação.

Chat Permite a comunicação entre doador e interessado para combinar a retirada.

Perfil Exibe informações do usuário e suas conquistas, como o Selo Eco.

Jornada do Usuário
O usuário acessa o aplicativo → navega pelos itens disponíveis → seleciona um item → entra em contato com o doador → combina a retirada → conclui a doação → recebe um selo como recompensa.
